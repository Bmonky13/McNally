Java Linear and Quadratic Equation Solver

Procedure

The first step is to make sure you have a Java JDK installed. You want to open files and navigate to "This PC", located in the left menu bar. Next double click on your "C:" drive, then double click on the "Program Files" foler. Inside here, locate and open the "Java" foler. (If this folder is not present move to the next bullet).

To install a Java JDK navigate to Oracle.com and download the appropriate installer for your operating system. If you are using windows, choose "Windows" in the tab bar and download the "x64 Installer". Next navigate to your downloads folder and double click on the installer. Allow the prompt to guide you through installation.once installation is complete, move to the next bullet.

Now, dowload my Quadratic Equation Solver Java file. Navigate to the location it was downloaded to and find the adress bar near the top left of the files app. For clarity, this adress bar is below the "Home Share View" buttons. Right lick anywhere in this adress bar and select "Copy address". Move to the next step.

Next, Open Command Prompt. You can acheive this by pressing the windows key and typing "cmd". Hit enter on your keyboard. A window should appear. Within this window type "cd " and press "Control" and "V" (on your keyboard) simultaneously to paste the path of the downloaded file that we copied earlier. The line of text you have typed should look something like: "cd C:\Users\m.north3\Downloads". Press enter. The text before your cursor should now look something like "C:\Downloads>" (note: this may look different depending on where you saved the file). If not, repeat the steps above.

Finally type "java Main" and hit enter. The program should start, and a lne of text will appear asking you to choose what type of equation you would like to solve. Allow my program to guide you through the solving of your equation.
